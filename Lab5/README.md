## 📘 Introduction
This lab explores how data moves across systems using different formats, conversion pipelines, a Master Data Management (MDM) merge, and a comparison of real‑time vs batch synchronization. You’ll work with four focused Python scripts that generate artifacts (JSON, YAML, XML, and logs) you can submit as screenshots.
